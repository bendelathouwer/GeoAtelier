# Things

Create the `Locations` before creating the Things, replace the xxx below with the `@iot.id` of the newly created Location

`https://<host>[:<port>]/FROST-Server/v1.0/Things`

```javascript
{
  "description": "Wemos D1 mini V3",
  "name": "DC3F7F",
  "properties": {
  },
  	"Locations": [
		{
			"@iot.id": xxx
		}
	]
}
```
