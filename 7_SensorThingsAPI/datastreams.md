# Datastreams

POST:
`http://<host>/FROST-Server/v1.0/Things(x)/Datastreams`

```javascript
{
  "name": "xxx",
  "description": "Datastream for xxx",
  "observationType": "http://www.opengis.net/def/observationType/OGC-OM/2.0/OM_Measurement",
  "unitOfMeasurement": {
    "name": "xxx",
    "symbol": "xxx",
    "definition": "xxxx"
  },
  "ObservedProperty":{"@iot.id": 000},
  "Sensor":{"@iot.id": 000}
}
```
